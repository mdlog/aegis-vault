## Overview

**Aegis Vault** is an AI-managed risk-controlled trading vault built on the 0G stack.

- **0G Chain** — Smart contract vault, policy, custody, and audit events
- **0G Compute** — AI agent inference for trading decisions
- **0G Storage** — State, decision journal, reasoning summary, and strategy memory

> Aegis Vault = verifiable AI risk manager with autonomous execution guardrails

## Getting Started

```bash
# Install dependencies
cd landing
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **Frontend:** React 19 + Vite 8 + Tailwind CSS 4
- **Charts:** Recharts
- **Routing:** React Router v7
- **Blockchain:** 0G Chain (Galileo Testnet)
